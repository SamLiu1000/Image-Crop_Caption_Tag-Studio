I hope this becomes your go-to tool for building training datasets.

Just click the `.bat` file, and it launches instantly in your browser.
It features three dedicated pages for:

* Image cropping
* Auto-captioning
* Manual tag management

The tool supports both English and Chinese, and is designed to be lightweight, fast, and easy to use.

All settings are saved directly in your browser, with options to export and import them—so you can switch between devices without losing your setup.

Getting started is simple: just install Python. No additional dependencies required.

---
### 🖼️ Image Cropper

Easily adjust the crop box size or apply your saved presets.
Zoom and move images freely to select the exact area you want.

* Automatically advances to the next image after cropping
* Import images from any folder
* Export `.txt` files to any location

---

### 📝 Image Captioner

Works seamlessly with local LLM tools like **LM Studio**, and supports OpenAI-compatible APIs from third-party providers.

* Fully customizable system and user prompts
* Generate captions or tags tailored to your needs

**Heads up:**

* Make sure **“Enable CORS”** is turned on
* If captioning stops midway on some images, try increasing the **context length**

---

### 🏷️ Tag Tool

A fast and flexible way to manage tags.

* Create quick tags organized by custom categories
* Add tags to images with a single click
* Use tag groups to apply multiple tags at once
* Built-in quick search for tags
